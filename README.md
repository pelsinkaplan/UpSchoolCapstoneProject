# Bazaar App
### UpSchoolCapstoneProject

This repo contains the source codes of my UpSchool Capstone Project. It is presented to the user as an example of an e-commerce application.

**Used Technologies**

- MVVM
-Coroutines
- Retrofit
- Hilt
- Singelton
- Data Binding
- Glide
- Firebase
- Room
- Live Data
- Fragments
- RecyclerView
- Navigation
- Toolbar
- Work Manager
- Firestore
- Lottie
